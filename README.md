## このサイトについて
本プロジェクトは、飲食店向けの提案・デモ用途として作成した汎用テンプレートです。  
店名・住所・電話番号・営業時間・メニューなどを差し替えるだけで、各店舗向けサイトとして利用できます。

## 主な機能
- レスポンシブデザイン（スマホ・PC対応）
- メニュー紹介
- 店舗情報・アクセス案内
- 予約導線（フォーム/電話）

## 注意
本テンプレート内の店名・住所・電話番号等はデモ用の仮情報です。  
本番利用時は各クライアント情報に置き換えてください。
