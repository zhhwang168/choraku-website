# 台湾料理 長楽 — 公式ウェブサイト

🌐 **网站地址**: [https://choraku-website.vercel.app/](https://choraku-website.vercel.app/)

> 🍜 岡崎市上和田町の台湾料理店「長楽」の公式サイトです。

---

## 🌐 サイトについて

愛知県岡崎市にある台湾料理店「長楽」のために制作したウェブサイトです。  
ラーメン・炒飯・定食・一品料理・宴会など、地域に愛される町の中華料理店の魅力を伝えます。

## ✨ 機能

- レスポンシブデザイン（スマホ・PC対応）
- Google フォームによるオンライン予約
- メニュー・店舗情報・アクセスページ
- 宴会・グループ利用案内

## 🛠️ 使用技術

- HTML / CSS / JavaScript（フレームワークなし）
- Google Forms（予約フォーム）
- Vercel（ホスティング）
